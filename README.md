# krisoye.github.io

Portfolio website for Krisoye Smith - Quantitative Researcher & Data Scientist

## About

This is my personal portfolio website showcasing my experience in quantitative finance, machine learning, and data science. The site highlights my work at Victory Capital, Macquarie Asset Management, CO-OP Financial Services, and Principal Global Investors, as well as open-source projects including `etlr` and `mdlr`.

## Featured Projects

- **etlr**: R package for streamlined ETL operations with S3 integration
- **mdlr**: R toolkit for model development and forecasting at scale
- **ETLRavenPackScores**: R package for processing RavenPack alternative data
- **ResearchR**: RAG system for research paper analysis using local LLMs
- **Credit Card Fraud Detection**: Real-time fraud prediction system (Featured)
- **EM Dynamic Stock Valuation Model**: MSCI Emerging Markets stock selection model
- **S&P 500 Volatility Signal**: Systematic volatility trading strategy
- **Global Equity Stock Selection**: $50B global equity model
